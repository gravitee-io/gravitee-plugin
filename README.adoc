== Description
The Gravitee.io plugin project contains a set of classes that are used in Gravitee.io products (APIM, AM, ...) to load and manage plugins. Several type of plugins are supported:

* Repositories
* Services
* Policies
* Identity Providers
* Notifiers
* Reporters
* ...
